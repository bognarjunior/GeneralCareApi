# GeneralCareApi

Back-end do aplicativo General Care — API para gerenciamento de saúde.

## 🚀 Tecnologias

- Node.js
- Fastify
- TypeScript
- Supabase (PostgreSQL)

## ⚡️ Como rodar

```bash
# Instale as dependências
npm install

# Crie um arquivo .env baseado em .env.example (quando disponível)
# Rode em modo desenvolvimento
npm run dev
